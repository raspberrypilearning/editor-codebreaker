<h2 class="c-project-heading--task">Decode the message</h2>
--- task ---
Decode your secret message using the .txt file
--- /task ---

**Run** your code so that it displays your encoded message. 

**Select** the encoded message and copy it. 

In `input.txt` delete your message so that it is empty. Next, **paste** your new message into the empty file. 

**Test:** Run your code and press 'c' when prompted. It will display the **decoded** version of your original message.
